# 菜鸟的coding · Day 001

猿神，起洞（分支）

2026-03-30 • C++ • acm.scu.edu.cn 15  ·  分支结构

题目

给定如下分段函数，输入浮点数 x（仅保留两位小数，|x| ≤ 10⁹），求 f(x) 并保留两位小数输出。

 f(x) = x − ⌊x⌋， 当 2k ≤ x ≤ 2k+1，k ∈ ℤ（整数部分为偶数）
 f(x) = ⌊x⌋ − x + 1，当 2k+1 ≤ x ≤ 2k+2，k ∈ ℤ（整数部分为奇数）

即：这是一个以 2 为周期的三角波函数，在偶数区间上升，在奇数区间下降。

思路

---

C++ · ape_cave.cpp 
#include <cstdio>
#include <cmath>

int main() {
    double x;
    scanf("%lf", &x);

    double k    = floor(x);       // ⌊x⌋，向下取整（正确处理负数）
    double frac = x - k;          // 小数部分，∈ [0, 1)
    long long ki = (long long)k;  // 整数部分，判奇偶

    double result = (ki % 2 == 0)
                 ? frac             // 偶数区间：x - ⌊x⌋
                 : 1.0 - frac;    // 奇数区间：⌊x⌋ - x + 1

    printf("%.2f\n", result);
    return 0;
}

// --- 验证 ---
// x = 0.59 → floor=0 (偶) → 0.59
// x = 1.73 → floor=1 (奇) → 1 - 0.73 = 0.27

复杂度分析

  • 时间复杂度：O(1)，单次运算。
  • 空间复杂度：O(1)。
